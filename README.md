

# Basil Web Application(vue)

This is the frontend for basil farm 


## Deployment

To deploy this project run

```bash
 
$ docker build -t basil_wa . 

$ docker run -p 8080:8080  -d basil_wa

```
## Develop

To deploy this project run

```bash
 
$ npm install

$ npm run dev


```
